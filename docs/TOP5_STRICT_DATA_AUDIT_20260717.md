## 审核结论

数据在当前本地库内通过了结构一致性审核：MatchID 外键、重复市场键和记录配对未发现异常。

但不能据此宣称“真实可靠100%”，原因是：

1. 归档原始来源的逐文件下载凭证和每个文件的独立哈希尚未全部登记；
2. 扩展市场 Provider 只有 `MarketAverage`，无法证明它代表哪些具体公司；
3. 当前数据主要提供初盘/收盘，不是每家公司连续时间序列；
4. 少数赛季存在少量市场记录缺口；
5. 尚未完成与官方赛果或第二独立来源的逐场交叉核验。

## 使用级别

- 1X2、亚洲盘、大小球：可进入研究和时间外回测；
- 多家公司同步走势：不可由该归档库单独推出；
- 正式盈利模型：未通过来源真实性和收益门禁；
- 单场截图：按用户确认的来源作为本场上下文证据，不能反推历史库全部真实。
