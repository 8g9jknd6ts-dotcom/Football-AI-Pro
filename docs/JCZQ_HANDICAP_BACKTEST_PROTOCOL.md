# 竞彩让球胜平负独立回测协议 v1.0

标准胜平负与竞彩让球胜平负完全分开。每场让球预测先生成主胜1球、2球、3球以上、平局、客胜的净胜球分布，再按实际主队让球数映射为让胜、让平、让负概率。

回测必须使用可审计的官方让球赔率历史，按 `LeagueCode, MatchDate, MatchID` 时间顺序逐场运行，并按 `LeagueCode, HomeHandicap` 分层报告。每场只能使用此前比赛；禁止未来数据泄漏。

赛前输出必须记录模型概率、去水后的市场隐含概率、edge、置信度与触发/拒绝原因。数据不足、盘口冲突、净胜球分布不清晰、edge不足或历史门禁未通过时，`HANDICAP_DECISION=SKIP`。

当前为 SHADOW_ONLY。近期单轮赛果只进入赛后审计，不能用于调参。
