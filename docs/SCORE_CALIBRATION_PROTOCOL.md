# AI评分、信心指数与冷门指数校准协议 v1.0

本协议在首次校准运行前固定。

## 样本

- 只使用已通过1X2基础门禁联赛的逐场滚动预测。
- 包含 JPN、SWE 与欧洲11个 validated 联赛。
- 每场预测均由比赛前历史数据生成；市场热门使用去水后的收盘平均赔率。

## 指标定义

### AI综合评分

`AI Score = max(PH, PD, PA) × 100`。

它表达模型所选方向的预测概率，不是历史命中率保证。

### 信心指数

`Confidence = clip((最高概率 - 第二高概率) × 200, 0, 100)`。

它是排序分，不直接解释为百分比命中率。

### 冷门指数

先确定收盘市场概率最高的方向，再计算：

`Upset Index = (1 - 模型赋给市场热门的概率) × 100`。

实际冷门定义为最终赛果不等于市场热门方向。

## 评价

- AI评分：二元Brier、10分箱ECE、最高四分位准确率提升。
- 信心指数：预测正确性的AUC、最高四分位准确率提升。
- 冷门指数：冷门事件的二元Brier、ECE与AUC。
- 同时报告逐联赛结果，防止总体样本掩盖联赛失效。

## validated门禁

### AI评分

1. 总样本不少于5,000；
2. ECE不高于0.05；
3. Brier优于常数基准；
4. 最高四分位准确率至少比总体高3个百分点；
5. 至少10个联赛同时满足 ECE≤0.08 且Brier优于常数基准。

### 信心指数

1. AUC不低于0.55；
2. 最高四分位准确率至少比总体高3个百分点；
3. 至少10个联赛满足 AUC≥0.52 且最高四分位提升为正。

### 冷门指数

1. AUC不低于0.55；
2. ECE不高于0.05；
3. Brier优于冷门基准发生率；
4. 至少10个联赛满足 AUC≥0.52 且Brier优于常数基准。

通过只代表指标定义得到历史校准支持，不代表可盈利或可进入 production。
