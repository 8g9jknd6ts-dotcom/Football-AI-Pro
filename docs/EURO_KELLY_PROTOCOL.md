# 欧洲赔率融合、返还率与凯利回测协议 v1.0

本协议在首次运行前固定。

## 样本与概率

- 只使用已通过1X2基础门禁的13个联赛：JPN、SWE及欧洲11联赛。
- 市场概率由 `MarketAverage` 收盘1X2赔率去水得到。
- 融合概率固定为：`75% × 去水市场概率 + 25% × 泊松概率`。
- 不根据本轮结果搜索融合权重。

## 选场与固定单位策略

1. 对每个方向计算 `EV = 融合概率 × 收盘赔率 - 1`。
2. 每场只选择EV最高方向。
3. 最高EV至少为3%才下注。
4. 固定单位策略每注1单位，用于计算ROI、标准误、95%下界和最大回撤。

## 凯利资金策略

- 完整凯利：`f = (p × odds - 1) / (odds - 1)`。
- 实际使用四分之一凯利：`0.25 × f`。
- 单场资金比例上限2%，下限0%。
- 每个联赛初始资金100单位，按当时资金复利下注。
- 报告最终资金与相对峰值最大回撤。

## candidate → validated研究门禁

每个联赛必须同时满足：

1. 至少500注；
2. 固定单位ROI大于0，且95%置信下界大于0；
3. 凯利最终资金大于100；
4. 凯利最大回撤不超过25%；
5. 逐场概率、赔率、返还率、EV、凯利比例和收益可复现。

即使通过也不得直接进入production：策略使用收盘平均赔率，实际成交时点和可得性未知。
