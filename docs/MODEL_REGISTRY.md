# 模型注册表

## 状态定义

- `candidate`：存在参数或结果声明，但尚未由本项目复现。
- `validated`：回测可复现并通过预设门禁，但尚未上线。
- `production`：通过回测、审查和版本发布，可用于正式推荐。

任何状态升级必须附：数据快照哈希、训练/验证时间切分、样本数、基准模型、Accuracy/Brier/LogLoss/ROI、赔率时间点、置信区间和可复现命令。

命中率优化协议：`docs/HIT_RATE_OPTIMIZATION_PROTOCOL.md`。当前所有高评分筛选仍属于 SHADOW 实验，必须同时报告覆盖率和弃权率，尚无 production 模型。

市场剥离诊断：`backtests/score-calibration-v1/MARKET_ABLATION.md`。当前AI与市场一致层命中率55.21%，冲突层27.64%；不能证明AI有独立超市场优势，AI冲突市场时默认降级或弃权。

可复现产物位于 `backtests/baseline-v1/`。当前没有任何 `production` 模型。
