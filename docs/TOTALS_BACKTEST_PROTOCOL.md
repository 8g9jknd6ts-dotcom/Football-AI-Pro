# 大小球策略回测协议 v1.0

本协议在首次大小球收益回测前固定。

## 市场与信息边界

- 使用 `TOTAL_CLOSE / MarketAverage / Line=2.5` 的大球和小球赔率。
- 球队强弱只使用当前比赛之前的完赛数据。
- 最近窗口10场，双方至少各5场历史。
- 泊松参数固定为主场系数1.15、rho=-0.12、最大比分8球。
- 当场赔率只用于价值选择与收益结算，不进入预期进球估计。

## 预注册策略

1. 计算 `Over 2.5` 与 `Under 2.5` 的模型概率。
2. 分别计算 `EV = probability × odds - 1`。
3. 选择EV较高的一侧，且EV至少为3%；否则跳过。
4. 每次固定下注1单位，不追损、不串关、不按凯利加注。

## candidate → validated 研究门禁

每个联赛必须同时满足：

1. 至少500注；
2. 实际ROI大于0；
3. 单注收益均值的95%正态近似置信区间下界大于0；
4. 逐场产物、MatchID、模型参数、数据与市场哈希齐全。

即使通过，也不得直接进入 production：收盘均价存在实际可得性和时间同步偏差，且本门禁只用于研究筛选。
