# 开盘—收盘市场迁移回测协议 v1.0

本协议在首次运行前固定。

## 定义

- 使用同一 `MarketAverage` 来源的 `1X2_OPEN` 与 `1X2_CLOSE`。
- 分别去除开盘与收盘水位，得到三方向公平概率。
- `Delta = CloseProbability - OpenProbability`。
- 选择Delta最大的方向；最大Delta至少为2个百分点才形成信号。

“市场迁移”只描述聚合赔率变化，不证明庄家主观意图、内幕信息或诱盘行为。

## 回测与结算

- 信号在收盘快照形成，使用对应收盘赔率结算。
- 每场最多一个方向，每次固定1单位。
- 同步报告信号准确率、开盘热门准确率、收盘热门准确率、ROI、收益标准误和95%下界。

## candidate → validated 研究门禁

每个联赛必须同时满足：

1. 至少500个信号；
2. 信号准确率至少比相同样本的开盘热门高2个百分点；
3. ROI大于0；
4. 单注收益95%正态近似置信区间下界大于0；
5. 开/收盘记录完整，MatchID与逐场产物可复现。

即使通过，也不能直接进入 production：MarketAverage 的收盘时点与真实下单可得性未知。
