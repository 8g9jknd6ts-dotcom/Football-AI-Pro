# 冷门猎手1.1候选升级

## 升级目标

冷门猎手不再把所有“市场热门不胜”混为一个事件，而是拆分为：

- 热门方不胜—平局型；
- 热门方不胜—对手获胜型。

## 新增输入

- 市场热门方及去水概率；
- 模型对热门方的概率；
- 多家公司数量与同向比例；
- 盘口移动是否逆热门方；
- 首发和伤停不确定性；
- 数据完整性。

## 输出

`Get-UpsetRiskProfile` 输出热门方不胜分、平局风险分、对手获胜风险分、风险类型、公司共识质量和首发不确定性。

所有分数目前都是候选风险排序指标，不是经过校准的真实概率，也不构成正式投注方向。

## 升级门禁

必须完成严格时间切分的1.0/1.1对照回测，并同时满足：

1. 总体Upset AUC提升且不低于0.55；
2. Upset Brier和ECE不恶化；
3. 至少10个联赛通过一致性条件；
4. 多公司字段缺失时不得把共识分数当作有效信号；
5. 任何收益门禁未通过时，仍只能作为风险提示。
