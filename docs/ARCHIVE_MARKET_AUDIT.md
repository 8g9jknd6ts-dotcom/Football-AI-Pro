# 大型归档市场数据审计

## 结论

归档文件名存在系统性污染，不能作为联赛身份依据。标准化过程只采用 CSV 内部 `Div` 字段、比赛日期、球队、比分和文件哈希。

典型问题：

- `bundesliga_2020_21.csv` 与 `denmark_2020_21.csv` 完全相同；内部均为 `D1`。
- 多个 Netherlands/Norway 文件完全相同；内部为 `N1`。
- 多个 Poland/Portugal 文件完全相同；内部为 `P1`。
- 21 个不同国家名称文件共享完全相同的40行样本，全部排除。
- `match_history_core.csv` 与15个分片是同一数据集的整体/切片形态，未重复导入。
