# Football AI Pro Shadow Backtest v1

状态：SHADOW / 不得用于正式盈利声明

用途：在挪超、巴甲等小样本联赛积累时间外样本，同时避免把短期波动误判为模型优势。

每场记录：MatchID、联赛、采集时间、模型版本、标准盘/让球盘、预测概率、推荐或弃权、出票赔率、最终赛果、命中、ROI、Brier、LogLoss、数据完整率。

门禁：

1. 采用滚动扩展窗口，不得随机打乱比赛。
2. 前100场仅观察，不升级模型。
3. 至少完成3个时间窗口、每个窗口不少于50场，才允许评估稳定性。
4. 以市场去水概率为基准，报告命中率差、Brier差、ROI和最大回撤。
5. 95%置信区间仍覆盖亏损时，保持SHADOW。
6. 所有不确定比赛允许弃权，弃权率与命中率一并报告。

小样本处理：联赛参数向全球基准收缩；样本越少，收缩越强。禁止用单场赛果或短期三连中作为升级证据。
