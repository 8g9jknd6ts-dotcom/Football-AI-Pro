# 世界杯竞彩分析框架 v2.1 - 本地部署指南

## 一、环境要求

- Python 3.8 或更高版本
- 操作系统：Windows / macOS / Linux 均可
- 磁盘空间：至少 200MB（数据库 50MB + 模型 100MB + 代码）

## 二、安装依赖

打开终端（命令行），执行以下命令：

```bash
pip install numpy scipy pandas scikit-learn lightgbm xgboost
```

如果安装失败，可以尝试：

```bash
pip install numpy scipy pandas scikit-learn
# lightgbm 和 xgboost 可选，如果安装失败可以跳过（ML模型部分会失效）
```

## 三、文件清单

将以下文件放到同一个文件夹中：

```
世界杯分析框架/
├── team_data.db                    # 数据库（143,200场比赛）
├── probability_to_goals_mapping.json  # 统计映射表

├── models/                         # ML模型文件夹（可选）
│   ├── intl_v2_direction_pre.pkl
│   ├── intl_v2_overunder_pre.pkl
│   └── ...
└── 部署说明.md                      # 本文件
```

## 四、快速开始

### 步骤1：运行分析脚本

```bash
cd 世界杯分析框架
python 本地部署分析框架.py
```

### 步骤2：修改比赛数据

打开 `本地部署分析框架.py`，找到文件底部的 `matches` 列表，修改为你想分析的比赛：

```python
matches = [
    {
        'name': '你想分析的比赛',
        'odds_h': 1.50,      # 主胜赔率
        'odds_d': 4.00,      # 平局赔率
        'odds_a': 6.00,      # 客胜赔率
        'handicap': -1,      # 让球数（-1=主队让1球，+1=主队受让）
        'hcap_odds_h': 2.00, # 让球主胜赔率
        'hcap_odds_d': 3.50, # 让球平赔率
        'hcap_odds_a': 2.80, # 让球客胜赔率
        'payout': 0.885,     # 返还率（竞彩0.885，欧洲0.93）
        'note': ''
    },
]
```

### 步骤3：再次运行

```bash
python 本地部署分析框架.py
```

## 五、核心功能说明

### 5.1 分析模块

| 模块 | 说明 | 输出 |
|------|------|------|
| 去水计算 | 赔率→真实概率 | 主胜/平局/客胜概率 |
| 统计映射 | 概率→预期进球 | 主队xG / 客队xG |
| DC泊松 | 预期进球→比分矩阵 | Top 5 比分概率 |
| 方向判断 | 胜负平概率 | 推荐方向 |
| 大小球 | 总进球预期 | 大/小球概率 |
| 半全场 | 半场xG模型 | 半场领先/平/落后概率 |
| 让球盘 | 净胜球概率 | 让胜/让平/让负概率 |
| CRI冷门 | 赔率结构检查 | 冷门信号 |

### 5.2 返还率设置

- **竞彩官方**：payout = 0.885（88.5%）
- **欧洲博彩**：payout = 0.93（93%）
- **威廉希尔**：payout = 0.95（95%）

不同返还率去水后的概率会有 2-4% 差异，但方向判断基本一致。

## 六、数据库使用（可选）

数据库 `team_data.db` 包含 143,200 场比赛，可以查询历史数据：

```python
import sqlite3
conn = sqlite3.connect('team_data.db')
cursor = conn.cursor()

# 查询某联赛的所有比赛
cursor.execute("SELECT * FROM match_history WHERE league_id = 'Premier League' LIMIT 5")
for row in cursor.fetchall():
    print(row)

conn.close()
```

### 6.1 数据库表结构

- `match_history`：俱乐部联赛（141,527场）
- `international_matches`：国际大赛（1,673场）
- `match_form`：近期状态数据
- `fifa_rankings`：FIFA排名数据

## 八、常见问题

### Q1：没有数据库文件怎么办？

可以只使用 `本地部署分析框架.py` + `probability_to_goals_mapping.json`，不需要数据库也能运行分析。数据库用于历史查询和ML训练，核心分析功能不需要数据库。

### Q2：安装 lightgbm/xgboost 失败？

这些库是ML模型需要的，如果安装失败：
1. 跳过ML模型部分（框架核心分析不依赖它们）
2. 或者使用 `pip install lightgbm --no-binary :all:` 从源码编译

### Q3：分析结果和之前不一致？

可能是因为：
1. 使用了不同的返还率（竞彩 vs 欧洲）
2. 赔率输入不同（初盘 vs 即时）
3. 让球盘理解有误（[-1] vs [+1]）

请检查 `payout` 参数和 `handicap` 符号是否正确。

### Q4：如何更新数据？

从 Football-Data.co.uk 下载最新赛季 CSV 数据，用 Python 导入数据库：

```python
import pandas as pd, sqlite3
df = pd.read_csv('2025-26_E0.csv')
conn = sqlite3.connect('team_data.db')
df.to_sql('match_history', conn, if_exists='append', index=False)
conn.close()
```
