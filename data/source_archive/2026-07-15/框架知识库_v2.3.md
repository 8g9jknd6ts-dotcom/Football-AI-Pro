## 一、核心认知

### 1.1 框架定位
本框架是一个**"基于赔率去水 + 统计映射 + DC泊松 + 多因子修正"** 的足球比赛分析系统，并非 AI 预测模型。框架本身不依赖机器学习，而是利用概率论和统计映射对赔率信息进行反推和修正。

### 1.2 两个分析模式

| 模式 | 目标 | 适用场景 |
|------|------|----------|
| **标准模式** | 预测最可能结果 + 防冷建议 | 日常分析 |
| **冷门猎手V1.0** | 反向视角，找被低估的高赔机会 | 发现陷阱、博高赔 |

**冷门猎手不是独立系统，而是叠加在基础框架上的反向解读镜片。两个模式共享底层数据，只是视角不同。**

## 二、基础框架（5个核心模块）

### 2.1 去水计算
- 竞彩返还率：88.5%（国内固定）
- 欧洲返还率：93%（William Hill 约95%）
- 公式：`真实概率 = (1/赔率) / 总概率和 × 返还率`
- 关键：去水概率归一化到1.0，不是0.885

### 2.2 统计映射（概率→预期进球）
- 使用 `probability_to_goals_mapping.json` 映射表
- 主胜去水概率 → 主队预期进球、客队预期进球、ρ系数
- 映射表通过 14 万+ 场历史数据拟合，不可手动修改
- 缺失区间用线性插值

### 2.3 DC泊松模型（预期进球→比分矩阵）
- 使用 Dixon-Coles 修正（ρ = -0.1 左右）
- 主场优势已内嵌在映射表的预期进球中
- 输出：各比分概率、方向概率、大小球概率、让球盘概率

### 2.4 CRI冷门识别引擎
- 检查赔率结构异常（平局赔率偏高、主胜集中度过高）
- 输出：0-10 分冷门分数
- 阈值：>6 分高风险，3-5 分中等风险，<3 分低风险

### 2.5 融合引擎（三层）
- 第一层：去水概率
- 第二层：DC泊松概率
- 第三层：统计映射对照
- 融合规则：去水概率为主，DC泊松修正偏差，统计映射验证量级

## 三、复盘因子库（关键！断档后恢复记忆的依据）

以下因子均来自实际比赛复盘，**每次分析必须检查是否适用**。因子不是每次都用，而是"条件触发"。

### 3.1 大败反弹因子（Big Loss Bounce）

**来源**：厄瓜多尔 0-0 库拉索（预期 2.15-0.45，实际 0-0）  
**教训**：1-7 惨败后球队往往背水一战，防守韧性 +30%  

```python
def big_loss_bounce(last_margin, opponent_strength):
    """大败反弹因子"""
    if last_margin <= -3 and opponent_strength > 0.7:
        # 惨败给强队后，本轮防守预期提升
        return 1.2  # 防守 +20%
    return 1.0
```

**适用条件**：
- 上轮净负 ≥ 3 球
- 对手是强队（或深盘获胜）
- 本轮是生死战/背水一战

**不适用**：
- 连续大败（反弹后可能再次崩溃）
- 球队已出线或淘汰无关

---

### 3.2 连胜疲劳因子（Winning Streak Fatigue）

**来源**：德国 2-1 科特迪瓦（10连胜后进攻预期过高）  
**教训**：连胜 ≥ 5 场 + 赛程密集，进攻预期应降低 10%  

```python
def winning_streak_fatigue(streak_length, days_since_last_match):
    """连胜疲劳因子"""
    if streak_length >= 5 and days_since_last_match < 4:
        return 0.9  # 进攻 -10%
    return 1.0
```

**适用条件**：
- 连胜 ≥ 5 场
- 距上轮比赛 < 4 天（赛程密集）
- 对手防守组织度好（非洲杯冠军级）

---

### 3.3 临阵换帅混乱因子（New Coach Chaos）

**来源**：突尼斯 0-4 日本（3天前换帅）  
**教训**：换帅后准备时间 < 7 天，不是"换帅如换刀"，而是战术混乱  

```python
def new_coach_chaos(days_since_appointment):
    """临阵换帅混乱因子"""
    if days_since_appointment < 7:
        return 0.75  # 进攻和防守都下降 25%
    elif days_since_appointment < 14:
        return 0.90  # 轻微影响
    return 1.0
```

**适用条件**：
- 教练上任 < 7 天
- 对手实力不差（日本体系球队）
- 不是长期磨合后的新帅红利期

---

### 3.4 日本体系修正因子（Japan Team Correction）

**来源**：日本 4-0 突尼斯（缺三笘薰+南野+久保，但替补爆发）  
**教训**：日本球员整体水平平均，核心缺失影响系数从 0.25 降至 0.05  

```python
def japan_team_correction(missing_core_players):
    """日本体系修正"""
    base_impact = len(missing_core_players) * 0.05
    return max(0.85, 1 - base_impact)  # 最多影响 15%
```

**适用范围**：仅日本国家队。其他球队核心缺失按 0.15-0.25 影响。

---

### 3.5 首战平后次战爆发因子（First Draw Burst）

**来源**：西班牙 0-0 佛得角 → 次战 4-0 沙特  
**教训**：首战被弱队逼平后，强队释放压力，次战上半场可能爆发  

```python
def first_draw_burst(team_strength, draw_opponent_strength, match_importance):
    """首战平后次战爆发因子"""
    if team_strength > 0.8 and draw_opponent_strength < 0.4 and match_importance == 'must_win':
        return 1.15  # 次战进攻预期 +15%
    return 1.0
```

**关键区分**：
- 首战 xG 高但被门将扑出（西班牙 0-0 佛得角 xG 2.5）→ 真实进攻能力未被否定
- 如果首战 xG 也很低 → 不是"被压制"而是"真不行"，不适用此因子
- 回归均值：超常表现只继承 30%（即 +15% 中 70% 是回归，只保留 4.5% 净提升）

---

### 3.6 对手含金量修正因子（Opponent Strength Adjustment）

**来源**：荷兰 5-1 瑞典复盘（瑞典 5-1 突尼斯是对手弱，不是瑞典强）  
**教训**：大胜后对手的防守强度必须重新评估  

```python
def opponent_strength_adjustment(opponent_historical_xg_conceded, opponent_actual_conceded):
    """对手含金量修正"""
    # 如果对手历史平均丢球 xG 很高，但本场丢很多 → 含金量低
    if opponent_actual_conceded > opponent_historical_xg_conceded * 1.5:
        return 0.85  # 大胜含金量打折，进攻预期下调
    return 1.0
```

**核心原则**：大胜弱队后，下一场对更强对手时，进攻预期不能按大胜比分线性外推。

---

### 3.7 xG vs 比分区分因子（xG Not Score）

**来源**：西班牙 0-0 佛得角（比分 0-0，但 xG 2.5）→ 次战 4-0 沙特  
**教训**：不能用比分直接推断状态，必须用 xG 区分运气因素和能力因素  

**核心原则**：
- 状态因子输入的是 **xG 差异**，不是 **比分差异**
- 0-0 但 xG 2.5 = 进攻能力还在，只是转化率为 0（门将神扑）
- 4-0 但 xG 2.5 = 超常转化，不能线性外推
- 回归均值：超常转化率 70% 会回归，只保留 30%

```python
def xg_based_state_correction(actual_score, expected_xg, opponent_goalkeeping_quality):
    """基于 xG 的状态修正（而非比分）"""
    if actual_score < expected_xg * 0.3:
        # 实际进球远低 xG → 可能是门将超水平，不是进攻不行
        return 1.0  # 不修正进攻预期，保持 xG 水平
    elif actual_score > expected_xg * 1.5:
        # 实际进球远超 xG → 超常转化，回归 70%
        excess = (actual_score - expected_xg) / expected_xg
        return 1.0 + excess * 0.3  # 只继承 30% 超常
    return 1.0
```

### 3.8 核心缺失影响分级（Core Absence Impact）

**来源**：多场复盘总结  

| 球队类型 | 核心缺失影响 | 说明 |
|---------|-------------|------|
| 日本体系 | 0.05/人 | 整体平均，替补水平接近 |
| 球星依赖型（如某队全靠1-2人） | 0.20-0.25/人 | 核心不可替代 |
| 欧洲顶级（西班牙/德国） | 0.10-0.15/人 | 体系完整但核心有提升 |
| 二线球队 | 0.15/人 | 核心与替补差距较大 |

**修正方式**：预期进球 × (1 - 核心缺失人数 × 影响系数)

## 四、冷门猎手V1.0（完整定义）

### 4.1 核心理念
传统分析追求"预测最可能结果"。冷门猎手反其道而行——**寻找市场共识与真实概率的偏离，挖掘高赔值博机会**。

冷门猎手不追求"正期望值"（竞彩 88.5% 返还率下所有单选项期望值均为负），而是寻找**相对价值最高**的选项——被低估程度最轻的结果。

### 4.2 模块一：赔率价值分析

**目标**：哪些结果被赔率低估  
**公式**：价值 = 真实概率 × 赔率 - 1  
**判断**：值 > 0 有价值（在竞彩几乎不可能），值越接近 0 被低估程度越低  
**实际用法**：在四个负值中，找最接近 0 的（即被抽水最少压缩的）

### 4.3 模块二：市场异动追踪

**目标**：庄家调盘的真实意图  
**方法**：对比初盘去水概率 vs 即时去水概率  
**信号阈值**：概率变动 > 3% 为显著信号  

| 变动方向 | 含义 |
|---------|------|
| 初盘→即时，热门概率大幅上升 | 可能有新信息（阵容/情报），或庄家诱盘 |
| 初盘→即时，冷门概率上升 | 庄家防范冷门，值得注意 |
| 变动 < 3% | 市场共识稳定，无特殊信号 |

### 4.4 模块三：冷门指数

**目标**：量化冷门可能性（0-100分）  
**评分逻辑**：

| 检查项 | 分数 | 条件 |
|--------|------|------|
| 高赔方概率 | 0-30 | 客胜/平局概率 > 15%：+30；>10%：+20；>5%：+10 |
| 标准盘热门程度 | 0-20 | 主胜 < 55%：+20；<65%：+15；<75%：+10；<85%：+5 |
| 返还率 | 0-15 | <90%：+15；<93%：+10；>93%：+5 |
| 历史同结构 | 0-20 | 浅盘(50-65%)：+20；中盘(65-80%)：+15；深盘(>80%)：+5 |
| 让球盘陷阱 | 0-15 | 穿盘率 < 45%：+15；<50%：+10 |

**阈值**：>60 高冷门风险；40-60 中等；<40 低

### 4.5 模块四：让球盘陷阱识别

**目标**：识别"标准盘热门≠让球盘稳胆"的陷阱  
**核心指标**：标准盘胜率 vs 让球盘穿盘率的差距

| 差距 | 陷阱评级 | 含义 |
|------|----------|------|
| > 25% | 🔴 高陷阱 | 强队大概率赢但大概率只赢1球或不赢 |
| 15-25% | 🟡 中陷阱 | 穿盘难度高于直觉 |
| < 15% | 🟢 低陷阱 | 穿盘率与胜率差距合理 |

**典型陷阱场景**：
- 标准盘主胜 66%（中盘热门），但 [-1] 让球盘穿盘率仅 44% → 差距 22% → 高陷阱
- 标准盘主胜 86%（深盘），[-2] 穿盘率 54% → 差距 33% → 但深盘特性，陷阱评级中等（因为深盘本身允许更大差距）

### 4.6 模块五：相对价值分析（竞彩 vs 欧洲）

**目标**：发现国内市场偏见带来的反向价值  
**方法**：竞彩去水概率 vs 欧洲真实概率对比

| 差异 | 含义 | 冷门猎手解读 |
|------|------|-------------|
| 竞彩主胜 > 欧洲主胜 + 2% | 国内更追捧主队 | 主队可能被过度定价，平局/客胜相对价值微升 |
| 竞彩客胜 > 欧洲客胜 + 2% | 国内更追捧客队 | 客队可能被过度定价 |
| 差异 < 2% | 无显著偏见 | 市场共识一致 |

## 五、已知模型缺陷与注意事项

### 5.1 泊松模型对极端比分的估计不足
- 当总预期进球 > 3.5 时，大比分（4-0、5-1）概率被低估
- 修正：生死战阶段使用负二项分布替代泊松，允许更大方差（dispersion=1.5）
- 世界杯第2轮生死战，进球率可能高于平均水平

### 5.2 心理因素无法完全量化
- 大败反弹、连胜疲劳、换帅混乱等因子只能做粗略修正
- 实际比赛中球员的心理状态、更衣室气氛、国家队荣誉感等无法数据化
- 因子修正幅度：±10-25%，不应过度自信

### 5.3 让球盘穿盘率的固有难度
- 四场比赛的让球盘穿盘率统计：54%、44%、40%、37%（均不过半或刚过线）
- 竞彩让球盘是**最难**的玩法，因为涉及"赢多少"而非"赢不赢"
- 建议：让球盘优先考虑双选覆盖，而非单选搏胆

### 5.4 单选期望值永远为负
- 竞彩 88.5% 返还率意味着所有单选长期期望值 = -11.5%
- 任何分析都无法改变这个事实
- 框架的价值是**提高方向命中率**，不是创造正期望值
- 正期望值只能通过组合套利（如两场都中）实现

### 5.5 xG 数据获取困难
- 大部分比赛没有公开的 xG 数据
- 西班牙 0-0 佛得角复盘能用到 xG 2.5 是因为有 Fox Sports 数据
- 没有 xG 时，用"射门次数 + 射正次数"粗略估算，精度下降

## 六、概率映射表校准参数

映射表 `probability_to_goals_mapping.json` 格式：
```json
{
  "0.075": [0.72, 2.91, 0.5],
  "0.125": [0.8, 2.39, 0.39],
  ...
}
```

每行含义：[主队预期进球, 客队预期进球, ρ系数]  
ρ系数：Dixon-Coles 修正参数，通常 0.3-0.6，主胜概率越高 ρ 越大（表示低比分概率被低估）

**校准记录**：
- 基础映射：14 万+ 场历史数据拟合（联赛为主）
- 国际大赛修正：DC 泊松参数针对国际大赛微调（国际大赛进球率通常比联赛低 15%）
- 但世界杯生死战可能进球率高于平均（背水一战）
- 具体修正：生死战 λ 方差增大，使用负二项分布（dispersion=1.5）

**下次校准时注意**：
- 如果预期进球与实际进球系统性偏差 > 20%，需要检查映射表区间
- 当前映射表在 0.525-0.575 区间（主胜概率 52.5%-57.5%）对应主队 1.62-1.83 球
- 西班牙 4-0 复盘验证：主胜概率 86.7% → 映射预期 3.44 球，实际 4 球（偏差 +16%，在合理范围）

## 七、使用流程（每次新会话）

1. 用户发送四个文件：
   - `team_data.db`（数据库，47MB）
   - `probability_to_goals_mapping.json`（映射表，1KB）
   - `本地部署分析框架.py`（脚本，10KB）
   - `框架知识库.md`（本文件，约 10KB）

2. 读取本文件，恢复所有复盘因子和冷门猎手定义

3. 读取数据库，确认数据表和记录数正常

4. 运行分析脚本，按标准流程执行：
   - 赔率去水 → 统计映射 → DC泊松 → 因子修正 → CRI冷门 → 融合输出
   - 如果用户要求"冷门猎手"，叠加 5 个反向解读模块

5. 每次分析结束后：
   - 如果比赛已结束，进行复盘
   - 发现新因子 → 更新本文件
   - 校准参数 → 更新映射表或脚本

### A.2 H2H 交锋表

**表名**: `head_to_head`  
**记录数**: 30,284 对（同联赛）  
**来源**: 从 match_history 预计算生成

**字段说明**：
| 字段 | 含义 | 示例 |
|------|------|------|
| team_a_id / team_b_id | 球队对（字母序统一） | ENG_Arsenal vs ENG_Liverpool |
| league_id | 联赛标识 | Premier League 2025-26 |
| total_matches | 交锋总次数 | 12 |
| team_a_wins / draws / team_b_wins | 胜负平统计 | 5-3-4 |
| team_a_goals / team_b_goals | 总进球 | 18-14 |
| team_a_home_wins / team_b_home_wins | 主客场胜 | 3-2 |
| last_5_results | 近5场序列（team_a视角） | WWDLW |
| first_match_date / last_match_date | 时间跨度 | 2019-08-10 至 2025-04-27 |
| avg_goals_per_match | 场均进球 | 2.67 |
| btts_rate | 双方进球率 | 58.3% |

**H2H密度最高联赛**：
- 埃及超：19.4 场/对（宿敌密度最高）
- 瑞士超：19.4 场/对
- 澳超：18.9 场/对
- K联赛：18.0 场/对
- 五大联赛：约 2.0 场/对（升降级导致不连续）

**使用方式**：分析时查询 `SELECT * FROM head_to_head WHERE (team_a_id=? AND team_b_id=?) OR (team_a_id=? AND team_b_id=?)` 获取交锋历史。

### A.3 数据完整性说明

**标准盘1X2赔率覆盖**：
- 五大联赛 + 次级联赛：99%+（B365/Avg/Max）
- 巴西 Serie A：2025-2026 有标准盘，2012-2024 仅让球盘
- 土耳其 Super Lig：100%（B365/Avg/Max）

**统计字段覆盖**：
- 射门/角球/牌：football-data 源联赛均有（英/德/法/西/意+次级）
- 半场数据：同上
- xG：⚠️ 数据库无此字段，需外部源（fbref/Understat）

**缺失数据**：
- 巴西 2012-2024 标准盘1X2（免费源无此数据）
- 所有联赛 xG（需付费API或外部抓取）
- 部分联赛早期赛季（2020前）因疫情数据不完整
